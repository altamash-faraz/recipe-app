## ğŸš€ **Quick Start**

### Prerequisites
- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/altamash-faraz/recipe-app.git

# Navigate to project directory

# Install dependencies

# or

# or

```

